Color OS运行第三方桌面及主题

发表于2023-07-28|更新于2025-02-04|技术系统coloros
|浏览量:

前言

众多周知，coloros系统会限制第三方桌面的使用,可为让用户深恶痛绝，今天博主发现了一个软件，可以解除这个限制。

教程

  • 首先在cts下载 这个网站下载一个名为Android x Rx CTS 验证程序 - ARM的压缩包，其中android x 为自己系统对应的安卓版本号。
  • 接着打开下载的压缩文件,找到一个名为cross profile test app的软件并安装。进入软件后点击中间open setting按钮即可解锁第三方桌面。此时软件会崩溃,正常现象。无需重启。
  • 然后就开始享受吧

注意事项_

必看-若无视警告至系统损坏与本人无关

  • 本软件仅在coloros系统测试过,其余系统可能出现未知bug
  • flyme系统严禁使用，会导致锁屏密码改变
  • 设置部分功能消失,可在搜索栏搜到
  • 设置部分界面变为类原生
  • 安装器变为类原生
  • 后台管理变为类原生 即划掉后台卡片并不能关闭软件
  • 部分设备会无法复原
  • 待补充

主题破解

免费试用主题后，打开设置-应用-应用管理-主题商店,让后点击停用按钮即可

复原

卸载软件重启即可
