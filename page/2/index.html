安卓正常运行瓦尔基里模组
发表于2023-07-28|更新于2025-02-04|技术我的世界|minecraft•mod•termux| 条评论
前言 众所周知，在安卓poajv里瓦尔基里mod会禁用物理引擎，导致玩家只能看着实体化的方块杵在原地。所以，为了解决这个问题，本教程诞生了。实际上本教程运用了一些取巧的方法。但确实做到了只用手机运行瓦尔基。 虽然瓦尔基里无法在安卓本地运行物理系统，但我发现，如果使用poajv与电脑端可联机，便可使用物理系统，于是我便试了在服务器中能不能运行瓦尔基里的物理系统，结果非常amazing，成功运行了。 原理 通过在手机上开服务器，pojav连接服务器来间接运行物理系统。（所以瓦尔基里实际上是运行在服务器上的，这也是我说取巧的原因） 教程准备：①一台性能足够的手机或两台性能一般的手机。（因为服务器如果和poajv运行在同一设备，那么压力会很大，所以建议将服务器单独放在另一设备运行）②termux app（用来运行服务器下载连接:）minecraft forge/fabric服务端（下载教程见下） 服务端下载:forge:前往此链接Forge下载 下载对应版本的forge安装器，随后在pojav安装jar文件
Color OS运行第三方桌面及主题
发表于2023-07-28|更新于2025-02-04|技术系统coloros|coloros•破解•主题•桌面| 条评论
前言众多周知，coloros系统会限制第三方桌面的使用,可为让用户深恶痛绝，今天博主发现了一个软件，可以解除这个限制。 教程 首先在cts下载 这个网站下载一个名为Android x Rx CTS 验证程序 - ARM的压缩包，其中android x 为自己系统对应的安卓版本号。 接着打开下载的压缩文件,找到一个名为cross profile test app的软件并安装。进入软件后点击中间open setting按钮即可解锁第三方桌面。此时软件会崩溃,正常现象。无需重启。 然后就开始享受吧 注意事项_必看-若无视警告至系统损坏与本人无关 本软件仅在coloros系统测试过,其余系统可能出现未知bug flyme系统严禁使用，会导致锁屏密码改变 设置部分功能消失,可在搜索栏搜到 设置部分界面变为类原生 安装器变为类原生 后台管理变为类原生 即划掉后台卡片并不能关闭软件 部分设备会无法复原 待补充 主题破解免费试用主题后，打开设置-应用-应用管理-主题商店,让后点击停用按钮即可 复原卸载软件重启即可
